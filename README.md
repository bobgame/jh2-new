# 开发服务器

运行 `npm ci` 安装依赖。

运行 `npm run serve` 启动开发服务器。

在浏览器中导航至 [http://localhost:4222/](http://localhost:4222/) 。

如果修改任何源文件，应用程序将自动重新加载。

# 构建

运行 `npm run build` 构建项目。
